17 [简答]简述模糊集合与经典集合的主要区别。

点击显示答案 经典集合中元素只能“属于”或“不属于”（隶属度为 0 或 1），而模糊集合允许元素以 [0,1] 之间的隶属度部分隶属。

18 [简答]简述模糊控制有哪些典型的应用领域。

点击显示答案 工业过程控制、家用电器（如空调、洗衣机）自动控制、机器人控制、汽车自动变速器等。

19 [简答]已知“接近于0的数”的隶属度函数为高斯函数 μ = e^(−x²/2)，则求 x = 0 和 x = 2 时，该隶属度的值。（注：^ 表示指数运算，例如 2² = 4）

点击显示答案 μ(0) = e⁰ = 1 μ(2) = e^(−2) ≈ 0.1353
